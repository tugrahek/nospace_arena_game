# 🚀 NoSpace: Arena

A fast-paced 2D arcade game focused on territory control and strategic movement.

> Capture space. Leave no room for the enemy.

---

## 🎮 Gameplay

- Move along the arena borders  
- Enter the field and draw lines  
- Close shapes to capture territory  
- Avoid enemies while drawing  

⚠️ If an enemy touches your active line, you lose a life.

---

## ⚡ Features

- 🧠 Adaptive enemy behavior (easy → challenging)
- ⚡ Area-based power-ups (unlock by capturing)
- 🔫 Shooting ability (via power-ups)
- 🧍 Multiple character styles
- 🌍 Arena themes
- 🏆 Combo & scoring system
- 📈 Progressive difficulty

---

## 🎯 Objective

Reduce enemy movement space and dominate the arena.
